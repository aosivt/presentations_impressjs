Актуальность

Катастрофический рост объемов информации в различнх сферах жизнидеятености таких как

  • Научная сфера;
  • Социальная сфера;

Целью исследования является:

Создание современной распределенной информационной системы, способной решить прикладные задачи дистанционного зондирования в пределах ограниченного времени.
Для достижения поставленной цели требуется решение следующих задач:
  • Составить аналитический обзор по теме исследования, включающий анализ программного обеспечения, предоставляющего набор инструментов реализации модулей/узлов для обработки данных дистанционного зондироваия;
  • Разработать основную концепцию и модель реализации поставленной цели;
  • Описать и реализовать созданную информационную систему обработки данных дистанционного зондирования;
  • Произвести развертывание созданной информационной системы в условиях реального применения.

Основные вегетационные индексы

№ Разновидность вегетационногоиндекса (ВИ) Формула и краткое описание
1. Относительный ВИ
(Ratio VI, RVI, Simple Ratio (SR))

$$RVI\eqby{}\frac{\rho_{NIR}}{\rho_{RED}}$$
2. Нормализованный разностный ВИ
(Normalized Difference VI, NDVI)

$$NDVI\eqby{}\frac{{\rho_{NIR}}-{\rho_{RED}}}{{\rho_{NIR}}+{\rho_{RED}}}$$
3. Трансформированный вегета- ционный индекс
(Transformed Vegetation Index TVI)

$$TVI\eqby{}\sqrt{NDVI+0.5}$$
4. Инфракрасный вегетационный индекс
(Infrared Percentage VI, IPVI)

$$IPVI\eqby{}\frac{{\rho_{NIR}}}{{\rho_{NIR}}+{\rho_{RED}}}$$
5. Разностный вегетационный индекс
(Difference VI, DVI)

$$DVI\eqby{}{\rho_{NIR}}-{\rho_{RED}}$$
